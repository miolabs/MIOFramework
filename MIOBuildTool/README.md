# miojs-tool

This is a cli tool to generate projects for [MIOJSLibs](https://github.com/miolabs/MIOJSLibs)

* [Usage](#usage)
* [Docs](#docs)
* [Changelog](#changelog)

## Usage

```bash
npm install miojs -g
miojs -h
```
